# OOP Order System

A simple .NET Console application built to practice core **Object-Oriented Programming (OOP)** concepts in C# through a small order management system.

## Goal
Apply OOP principles in a practical project instead of only learning them theoretically.

## Concepts Covered

### OOP Principles
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

### C# Features
- Abstract & Sealed Classes
- Method Overloading
- Extension Methods
- Static Classes
- init Accessor
- Type Casting
- out Keyword & TryParse

### Design Principles
- SRP (Single Responsibility Principle)
- Modular Design

### Other
- JSON Serialization
- Random Number Generation
- Object Base Class usage

---

## Project Description

This project simulates a simple order management system for a small online store.

### Features
- Define different product types
- Create orders containing multiple products
- Calculate final order price
- Apply different discount strategies
- Store and load data using JSON
- Generate simple order reports

The system is intentionally kept simple but designed to be **clean, maintainable, and easily extendable**.
